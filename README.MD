โปรเจคนี้เป็นโปรเจคเกี่ยวกับการจำแนกใบหน้าผ่านกล้อง โดยใช้ Library face_recognition ซึ่งก็คือการค้นหาใบหน้าของบุคคลจากภาพหรือวีดีโอมาทำการประมวลผล
และมีการจดจำใบหน้าผ่านการ ถ่ายภาพจาก OpenCV เก็บไว้เป็น dataset ใช้ในการจดจำและจำแนกบุคคลที่ทำการบรรทึกข้อมูลหน้าตนเองแล้ว
เมื่อทำการนำใบหน้าเข้ามาใน frame กล่องก็จะตรวจจับคัดแยกว่าบุคคลนี้คือใครและสามารถบรรทึกการตรวจจับได้ว่าบุคคลนี้ได้ทำการแสกนตรวจจับใบหน้า ณ เวลานั้นแล้วลงไฟล์ CSV
